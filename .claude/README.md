# Claude Project Directory

This directory contains contextual information and instructions for Claude when working on the DeepSOW project. It helps Claude understand the project's goals, architecture, coding standards, and provides reusable task templates and custom skills.

- `project.md` – Project overview, problem statement, target audience.
- `rules.md` – Coding conventions, tech stack details, preferences.
- `tasks/` – Task definitions for common activities (feature implementation, bug fixing).
- `skills/` – Specialized instructions for performing specific functions (e.g., SOW parsing with OpenAI).

Use this structure to maintain consistency and speed up development with Claude's assistance.
